미디어 쿼리 연습

미디어쿼리는 화면 크기에 따른 각각의 속성값을 지정하여 여러가지 화면을 구성하는 기술입니다

@media only all and (조건문) {실행문}

  • @media : 미디어쿼리가 시작됨을 표시합니다.
  • only : 미디어쿼리 구문을 해석하라는 명령입니다.(생략가능)
  • all : 미디어쿼리 해석해야할 대상을 나타냅니다.(생략가능)
    • all : 모든 미디어 유형에서 사용할 CSS를 정의합니다.
    • print : 인쇄 장치에서 사용할 CSS를 정의합니다.
    • screen : 컴퓨터 스크린 장치에서 사용할 CSS를 정의합니다.
    • aural : 화면을 읽어 소리를 출력하는 장치에서 사용할 CSS를 정의합니다.
    • tv : TV에서 사용할 CSS를 정의합니다.
    • handheld : 손에 들고 다니는 장치에서 사용할 CSS를 정의합니다.
    • project : 프로젝트장치에서 사용할 CSS를 정의합니다.
  • and : 앞과 뒤의 조건을 나타냅니다.(생략가능)
  • (조건문) : 해당조건을 설정할 수 있읍니다.
  • {실행문} : 조건에 따른 실행을 설정합니다.
